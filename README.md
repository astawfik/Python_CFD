# Numerical Solvers from Scratch in Python

This repository contains my independent implementations of the computational fluid dynamics (CFD) solvers and numerical methods outlined in the [**Python_CFD**](https://drzgan.github.io/Python_CFD/intro.html) course by Prof. Zhengtao Gan. 

The objective of this project was to transition from basic 1D advection models to solving the full 2D incompressible Navier-Stokes equations from scratch using Python, without relying on high-level black-box simulation wrappers.
